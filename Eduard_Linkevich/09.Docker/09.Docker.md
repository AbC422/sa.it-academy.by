### Dockerfile
```bash
FROM python:3.8
MAINTAINER Eduard Linkevich <edlink911@gmail.com>

WORKDIR /app

COPY ./requirements.txt ./
RUN pip3 install -r ./requirements.txt

COPY . /app
WORKDIR /app

ENTRYPOINT python main.py
```

### main.py
```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return 'Hello World'

if __name__ == "__main__":
    app.run(host='0.0.0.0')
```
