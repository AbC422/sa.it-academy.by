
## Adding user ansible with superpowers
```bash
ansible work_sa -m shell -a "adduser ansible"
ansible work_sa -m shell -a 'echo "ansible ALL=(ALL:ALL) NOPASSWD: ALL" >> /etc/sudoers'
```
